HTML

  • Hypertext Markup Language (HTML): used to create structure of webpage
  • head: element contains information about the webpage
  • body: element represents the visible content shown to the user
  • !DOCTYPE html: declaration of document type, not case sensitive
  • lang "en": english language
  • style: defines CSS style of website
  • base: defines base URL for webpage
  • link: connects external resource to HTML doc
  • h1, h2: heading elements for given text block
  • img: element containing image displayed on webpage
  • href: points to URL for link
  • id: way to identify elements and select them

CSS

  • Cascading Style Sheets(CSS): used to define how webpage content is viewed, layout
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS: used directly in HTML, use style attribute with CSS rule
  • Internal CSS: embed entire style sheet directly in HTML, rules only apply to that page
  • External CSS:separate file of CSS rules, link used to add to HTML, most common
  • selector (div): defines element or attributes to which the rules, declarations, apply
  • declaration (color:blue): contains the CSS property and the value of the property

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and swtiches to it
  • cd: navigates between directories in path
  • git add -A: stages changes made and commit all files

Javascript

  • JavaScript: adds interactivity to website
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • string: sequence of text enclosed in single quote marks 'Bob'
  • Boolean: true/false value
  • Array: structure that allows you to store multiple values in a single reference
  • "if" function: checks if statement follows condition
  • "for" loop: uses predictable pattern of indices to perform task on all items in array over and over
  • starting point: variable declared and value given to begin array
  • condition: number has to be less than length of array for loop to continue
  • final statement: allows the loop to repeat, determines what variable will change by
